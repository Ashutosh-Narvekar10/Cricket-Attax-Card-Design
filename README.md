 Cricket-Attax-Card-Design

Hi Guys! 
Being passionate about cricket I have created a web-based card generator for cricket enthusiast like me. 
---
##  Features
- Custom Card Creation : Users can enter player details (name, specialty, stats, etc.) and generate a personalized card.
- Country Flag Integration : Automatically fetches and displays the player’s country flag using the REST Countries API.
- Star Rating System : User inputs a number, and the card shows that many ⭐ stars dynamically.
- Player & Team Images : Supports adding images via URL (player photo + IPL team logo).
- Fantasy Points & Stats Display : Shows key player stats like matches, runs, wickets, averages, etc. on the card.
- Background Removal : Player images can have backgrounds removed using the Remove.bg API.
- Error Handling : Alerts if invalid country names are entered or if background removal fails, with fallbacks.
